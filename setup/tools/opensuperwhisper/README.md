# OpenSuperWhisper

OpenSuperWhisper is a macOS voice dictation app.
Use it to speak prompts into the Codex CLI running inside WezTerm and tmux.

## Install

Install OpenSuperWhisper with Homebrew:

```bash
brew install --cask opensuperwhisper
```

Open it:

```bash
open -a OpenSuperWhisper
```

## Usage

Configure a global hotkey in OpenSuperWhisper.
Focus the Codex composer in WezTerm.
Use the hotkey to dictate a prompt into the composer.
Review the dictated text before pressing Enter.

Voice dictation is input assistance only.
It should not bypass prompt review for destructive, expensive, or high-risk tasks.
